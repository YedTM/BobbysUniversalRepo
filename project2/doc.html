Dog Documentation

The API used for this project was the Dog.API

The font used throughout the entire project is Bree Serif.

I've had many issues with CORS, which seem to occur due to either not being on a live server or because of the way certain links were formatted. This has been fixed now.

The No Image image was found here.

The last searched primary breed is kep track of so that the user has the same breed ready to be generated again.

The three controls include: Two drop downs for the requested breed, one drop down for the amount of images generated, and the last allows the user to select images and save them to a favorites page.
